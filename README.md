## How to install and run locally ?

```
$ git clone https://github.com/sanket4120/queezly.git
$ cd queezly
$ npm install
$ npm start
```
## Features -

- Quiz by category
- Rules Page
- MCQ Based Question 
- Quiz Result 
- User Profile 
- Timer for Questions
- Authentication
      - User Signup
      - User Login
      - User Logout
- Responsive

## Built with -
- ReactJS
- [Modulo](https://modulo.netlify.app/) component library
- [Mockbee](https://mockbee.netlify.app/) for backend
- React Router
- React Context API
